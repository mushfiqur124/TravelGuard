🔧 Troubleshooting

If the context menu doesn't appear:

  • Make sure you've highlighted/selected the text first
  • Check that the TravelGuard extension is enabled in Chrome extensions
  • Refresh this page and try again
  • Open Chrome Developer Tools (F12) to check for any console errors
